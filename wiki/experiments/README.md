# Experiments

This directory holds structured records of hypotheses, tests (backtests or live paper), results, and lessons.

Format for each experiment (create as `YYYY-MM-slug.md` or use subdirs):

- Hypothesis
- Method / parameters
- Data used (time range, markets)
- Results (quant + qualitative)
- Decision / next action (promote, iterate, discard)
- Links to journal entries or reflections
