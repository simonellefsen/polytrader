# Decisions

This directory records important, durable decisions with context, alternatives, and (retrospectively) outcomes.

Each decision file should contain:
- Date decided
- Problem / context
- Options considered
- Decision + rationale
- Consequences / follow-ups
- (Later) Review date + outcome
